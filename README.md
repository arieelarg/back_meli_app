# Descripción
Api para la visualización, consulta, búsqueda de alertas, y estadística. 

# Tener instalado
<ul>
    <li>Node.js V12+</li>
    <li>npm V6+</li>
    <li>Docker</li>
</ul>

# ¿Cómo iniciar el proyecto?
```bash
git clone https://github.com/arieelarg/back_meli_app
cd ./Docker
docker-compose up
cd ..
npm i && npm start
```
